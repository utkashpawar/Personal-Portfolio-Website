# Personal Portfolio Website

This is my personal portfolio website built as part of a web development assignment. It showcases my skills, projects, and contact information using basic HTML.

## About

A simple, structured personal portfolio page built using semantic HTML — no CSS or JavaScript, focused on correct use of headings, paragraphs, lists, and links.

## Sections

- About Me
- Skills
- Interests
- Projects
  - Online Library Management System (PHP, MySQL, SQL, HTML, CSS, XAMPP)
  - Restaurant Menu Management Web Page (HTML, CSS, JavaScript)
- Contact Information

## Tech Used

- HTML5

## How to Run

1. Clone this repository
   ```
   git clone https://github.com/utkarshpawar/portfolio-website.git
   ```
2. Open `index.html` in any browser
