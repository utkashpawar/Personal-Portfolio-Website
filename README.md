# Personal Portfolio Website

This is my personal portfolio website, built as part of the VaultofCodes web development internship (Assignment 1). It showcases my background, skills, interests, projects, and contact information using semantic HTML.

🔗 **Live demo:** https://utkashpawar.github.io/Personal-Portfolio-Website/

## About

A simple, structured personal portfolio page built using semantic HTML — no CSS or JavaScript, focused on correct use of headings, paragraphs, lists, and links as outlined in the assignment brief.

## Sections

- **About Me** — background as a final-year IT student at Sant Gadge Baba Amravati University, aspiring software developer
- **Skills** — Java, Python, SQL, HTML, CSS, JavaScript, React.js
- **Interests** — Web Development, Database Management, Cloud Computing, Competitive Coding and Hackathons
- **Projects**
  - **Online Library Management System** — PHP, MySQL, SQL, HTML, CSS, XAMPP. User authentication with PHP sessions, CRUD operations, and a responsive UI to display books and user info.
  - **Restaurant Menu Management Web Page** — HTML, CSS, JavaScript. Dynamic add/remove of food items in real time via DOM manipulation, no page reload.
- **Contact Information** — email, LinkedIn, and GitHub links

## Tech Used

- HTML5 (semantic structure: `<section>`, `<h1>`–`<h3>`, `<ul>`, `<a>`)

## How to Run

1. Clone this repository
   ```bash
   git clone https://github.com/utkashpawar/Personal-Portfolio-Website.git
   cd Personal-Portfolio-Website
   ```
2. Open `index.html` in any browser — no build step or dependencies required.

Deployed with [GitHub Pages](https://pages.github.com/) — auto-builds from the `main` branch.
